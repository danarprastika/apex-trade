# QuantX AI

Production-grade personal AI trading platform.

## Architecture

This project follows Clean Architecture, DDD, and SOLID principles.

## Quick Start

```bash
# Start development environment
docker compose -f docker-compose.dev.yml up -d

# Run backend tests
cd backend && pytest tests/

# Run frontend dev server
cd frontend && npm run dev
```
